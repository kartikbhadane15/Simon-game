🎮 Simon Says Game
A classic Simon Says Memory Game built using HTML, CSS, and JavaScript. The game challenges players to remember and repeat an increasingly long sequence of colors generated by the system. As the game progresses, the sequence becomes more complex, testing the player's memory and concentration skills.

🚀 Demo
Press any key to start the game and follow the color sequence shown by the computer. Repeat the sequence correctly to advance to the next level. If you make a mistake, the game ends and your score is displayed.

✨ Features
Interactive memory-based gameplay
Random color sequence generation
Progressive difficulty with each level
Visual feedback for game and user actions
Game-over detection and score display
Ability to restart the game after losing
Simple and responsive user interface

🛠️ Technologies Used
HTML5 – Structure of the game
CSS3 – Styling and animations
JavaScript (ES6) – Game logic and interactivity

🎯 How to Play
Open the game in your browser.
Press any key to start.
Watch the color sequence displayed by the game.
Repeat the sequence by clicking the colored buttons.
Each correct round adds a new color to the sequence.
Continue playing until you make a mistake.
Your final score will be displayed when the game ends.

🧠 Game Logic
The game generates a random color sequence.
The player must reproduce the exact sequence.
After every successful level:
The player's input sequence is reset.
A new color is added to the game sequence.
The difficulty increases progressively.
If the player's input does not match the generated sequence, the game ends and resets.
